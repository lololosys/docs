## Docswrite WordPress Plugin

The Docswrite WordPress plugin enhances your publishing workflow by providing advanced features and better integration with your WordPress site. It's designed for editorial and content teams who want to automate and optimize their content publication process.

## Troubleshooting

### Common Issues

**Connection Problems**

- Ensure you're logged into Docswrite when connecting
- Check that your WordPress site is accessible from the internet
- Verify your permalink structure is not set to "Plain"

**SEO Settings Not Working**

- Confirm you have Yoast SEO or RankMath installed
- Check that the plugins are activated and up to date
- Verify you're using the latest version of the Docswrite plugin

**Publishing Failures**

- Check your WordPress user permissions
- Ensure the plugin is activated
- Verify your Docswrite account connection

### Getting Help

If you encounter issues:

1. Check the plugin status in WordPress admin
2. Review error messages in the Docswrite dashboard
