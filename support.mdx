## What to Include in Your Support Request

To help us resolve your issue quickly, please include:

### For Technical Issues

- **Error messages**: Exact text of any error messages
- **Steps to reproduce**: What you were doing when the issue occurred
- **Browser/device**: What browser and operating system you're using
- **Google Doc URL**: If the issue is with a specific document
- **WordPress site**: Your WordPress site URL (if relevant)

### For Account Issues

- **Account email**: The email address associated with your Docswrite account
- **WordPress site**: The WordPress site you're trying to connect
- **Subscription details**: Your current plan and billing information (if relevant)

### For API Issues

- **API endpoint**: Which endpoint you're trying to use
- **Request details**: The complete request you're making
- **Response received**: The actual response you're getting
- **Expected behavior**: What you expected to happen

## Common Issues & Quick Fixes

### Google Drive Connection Issues

- Ensure you've granted all required permissions
- Try disconnecting and reconnecting your Google Drive
- Check that your documents are shared with "Anyone with the link"

### WordPress Connection Issues

- Verify your WordPress site URL is correct
- Check that your WordPress site is accessible
- For self-hosted WordPress, ensure application passwords are enabled

### Publishing Failures

- Confirm your Google Doc is shared properly
- Check that your WordPress connection is active
- Verify you have publishing permissions on your WordPress site

### API Issues

- Ensure you're using the correct endpoint URL
- Check that your API token is valid
- Verify all required parameters are included
